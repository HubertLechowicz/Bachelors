# hidenseek

---

### How to run

1. Create `virtualenv`
2. Activate virtual environment
3. Install dependencies from `req.txt`
   > `pip install -r req.txt`
4. Start game by using `python main.py`
5. If there are some errors related to local `hidenseek` library, use `pip install -e .`

---

### Dependencies

This game has been built by using `pygame<2.0.0`. Installing newest version may break some mechanics.
